============booking status============
pending ==> user want to book but not yet payment_id
confirm ==> user already paid but not check_in
check_in ==> user arrived and stay at hotel
check-out ==> user check_out and gone
cancelled ==> user make a booking and cancel before payment done


=============room status============
Available ==> room is Available
Reserved ===> room is booked by guest
Occupied ==> guest already in room(during staying)
Maintainance ==> room is under maintainance
